Gerador de Senhas em Python

Projeto simples desenvolvido para praticar Python, utilizando funções, listas, loops e módulos da biblioteca padrão para gerar senhas aleatórias seguras.

Funcionalidades
Gera senhas aleatórias de qualquer tamanho
Utiliza letras maiúsculas e minúsculas
Inclui números
Inclui caracteres especiais
Valida o tamanho mínimo da senha
Embaralha os caracteres para aumentar a aleatoriedade
Tecnologias
Python 3
Módulo random
Módulo string
Como executar
Clone o repositório
git clone https://github.com/eric-domingues1/gerador-de-senhas-python.git
Acesse a pasta do projeto
cd gerador-de-senhas-python
Execute o programa
python gerador.py
Exemplo de uso

Entrada:

Digite o tamanho da senha: 12

Saída:

@A8m#2Qx!7Lp
