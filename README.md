# Gerador de Senhas em Python

Projeto simples desenvolvido para praticar Python.

## Funcionalidades

- Gera senhas aleatórias
- Utiliza letras maiúsculas e minúsculas
- Inclui números
- Inclui caracteres especiais
- Valida tamanho mínimo da senha

## Tecnologias

- Python 3
- random
- string

## Como executar

```bash
python gerador.py
```
